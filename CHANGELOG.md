# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-22
### Added
- **Phase VI**: Batch Processing support for processing entire directories.
- **Phase VI**: Dockerization with `Dockerfile`.
- **Phase VI**: Modern packaging with `pyproject.toml` and `setup.py`.
- **Phase V**: Multi-algorithm engine (Wiener, Spectral Subtraction, LMS).
- **Phase V**: Real-time audio metrics (RMS, ZCR, Centroid).
- **Phase IV**: Advanced Wiener filtering and VAD support.
- **Phase III**: STFT/ISTFT architecture and WAV I/O.
- **Phase II**: High-fidelity HTML Dashboard with real-time Web Audio API.
- **Phase I**: Initial Spectral Gating engine.
